### 핵심 기능

- 타임박싱 불편해.. 좀 더 몇 번 클릭 없이 바로 입력됐으면 좋겠다
	- 지금의 구글 캘린더: 매번 텍스트 입력, 시간 설정, 색깔 설정 해야 함. 귀찮음.
	- => 기존에 이미 입력되어 있던 Task의 경우 바로 선택해서 입력할 수 있었으면 좋겠음
	- 위젯에서 쉽게 보고 바로 기록 누를 수 있었음 좋겠다
- 몰입이 필요한 시간에는 실제 몰입한 시간만 기록되었으면 좋겠다
	- 비쥬얼타이머처럼 중간중간 멈췄다, 다시 시작했다 할 수 있게 되었으면 좋겠다
	- 몰입의 시간 체크하면 라이브액티비티 떠서 -> 바로 비쥬얼타이머 시작될 수 있게
- 내 시간 활용에 대한 통계, 인사이트, 그리고 더 낫게 쓸 방법 제안해줬으면 좋겠음
	- 이번주 얼마나 몰입했는지, 이번달 얼마나 몰입했는지, 얼마나 시간을 밀도있게 썼고 얼마나 시간을 허비했는지 자각할 수 있게

### 부가 기능
- AI가 지금까지의 통계를 기반으로 타임박싱 제안 -> 확정 또는 수정 후 확정 또는 버리기 등
